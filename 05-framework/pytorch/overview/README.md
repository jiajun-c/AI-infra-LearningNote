# 整体框架

## 1. 整理框架

```shell
┌─────────────────────────────────────────────────────────────┐
│                    Python 层 (torch/)                        │
│  - __init__.py (入口)                                        │
│  - nn/ (神经网络模块)                                         │
│  - optim/ (优化器)                                           │
│  - distributed/ (分布式训练)                                  │
│  - _inductor/ (编译器后端)                                   │
│  - _dynamo/ (图捕获)                                         │
│  - backends/ (后端配置)                                      │
├─────────────────────────────────────────────────────────────┤
│                   扩展模块层 (torch/_C)                       │
│  - torch/_C/__init__.pyi (类型定义)                          │
│  - torch/csrc/Module.cpp (Python 绑定)                       │
├─────────────────────────────────────────────────────────────┤
│                   ATen C++ 层 (aten/)                         │
│  - aten/src/ATen/ (核心张量库)                               │
│  - Context.h/cpp (全局上下文)                                │
│  - native/ (算子实现)                                        │
│  - cuda/hip/ (GPU 后端)                                      │
├─────────────────────────────────────────────────────────────┤
│                   底层依赖层                                  │
│  - c10/ (核心抽象：Tensor、Device、Allocator)                 │
│  - CUDA/cuDNN/cuBLAS                                         │
│  - MKL/OneDNN (CPU 加速)                                     │
└─────────────────────────────────────────────────────────────┘

```
