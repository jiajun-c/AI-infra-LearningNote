# Torch框架

## 1. 整体框架

整体框架大致分为python前段 -> python绑定层 -> Aten层 -> C10 -> 各类加速后端

```shell
┌─────────────────────────────────────────────────────────┐
│              Python Frontend (torch/)                    │
│  - torch.nn (神经网络模块)                                │
│  - torch.autograd (自动微分)                             │
│  - torch.distributed (分布式训练)                         │
│  - torch._inductor / torch._dynamo (TorchCompile)        │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│              torch/csrc (Python 绑定层)                   │
│  - THP (Torch Python) 绑定                              │
│  - Python 对象与 C++ 对象的桥接                           │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│              ATen (A Tensor Library)                     │
│  - 张量操作的 C++ 接口                                     │
│  - 原生算子实现 (native/)                                │
│  - 设备特定实现 (cuda/, cpu/, mps/)                      │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│              c10 (Caffe2 核心)                            │
│  - TensorImpl (张量底层实现)                              │
│  - StorageImpl (存储实现)                                 │
│  - 分配器 (Allocator)                                    │
│  - Device/Stream 管理                                    │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│              后端加速库                                   │
│  - CUDA/cuDNN (NVIDIA GPU)                               │
│  - ROCm/MIOpen (AMD GPU)                                 │
│  - oneDNN/MKL (Intel CPU)                                │
│  - MPS (Apple Silicon)                                   │
└─────────────────────────────────────────────────────────┘

```
