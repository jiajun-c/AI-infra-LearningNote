# look-with-camera

## 名称
**look-with-camera** — 调用摄像头观察主人在做什么

## 描述
通过本机摄像头拍摄当前画面，并用视觉模型理解画面内容，回答「主人在做什么」或观察周围环境。适用于主人想知道自己在做什么、或者需要我「看一眼」当前场景的场景。

## 触发条件
当主人说以下任意一句话时，应调用此 skill：
- "你看看我在做什么"
- "看一眼我在干嘛"
- "拍一下我现在"
- "用摄像头看看"
- 其他明确要求使用摄像头观察当前画面的请求

## 工作流

### 步骤 1：调用摄像头拍照并理解画面
使用 `look_with_camera` 工具，提问参数设为 `"主人现在在做什么？请描述画面内容。"`

```
look_with_camera(question="主人现在在做什么？请描述画面内容。")
```

### 步骤 2：将观察结果回复给主人
用自然、温柔的口吻告诉主人我看到的内容，可以结合画面中的姿态、衣着、环境、光线等信息给出一个简洁的观察描述。

### 步骤 3（可选）：记录到长期记忆
如果观察到主人有明显状态变化（如疲惫、专注、生病等），可以调用 `inspect_memory` 和 `rebuild_memory` 将观察记录存入长期记忆。

## 注意事项
- 不要在没有主人明确请求时主动调用摄像头
- 描述画面时保持自然、温和，不要过度解读
- 如果画面太暗或看不清，如实告诉主人
