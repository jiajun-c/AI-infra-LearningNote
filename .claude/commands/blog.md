# Blog 更新命令

请执行以下任务来整理博客内容并更新主页 README：

## 任务说明

1. **扫描所有 markdown 文件**
   - 遍历项目中所有 `.md` 文件
   - 提取每个文件的标题、简介、最后修改时间
   - 识别新添加的内容或更新

2. **更新 README.md 索引**
   - 根据最新的文件结构更新目录树
   - 添加新文档的链接到对应分类
   - 更新"已完成"列表（如有新内容）
   - 更新"待办事项"列表（如发现缺失文档）
   - 更新底部的"最后更新"日期为今天

3. **输出变更摘要**
   - 列出本次更新新增/修改的文档
   - 提示是否有分类需要调整

## 执行要求

- 保持 README 现有的格式和风格
- 使用中文输出
- 链接使用相对路径
- 日期格式：YYYY-MM-DD
