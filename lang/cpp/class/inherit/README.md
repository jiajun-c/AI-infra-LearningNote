# C++ 类继承

## 1. 继承基础

在 C++ 中，继承允许一个类（派生类）继承另一个类（基类）的成员。

```cpp
class Base {
public:
    void baseMethod() {
        // 基类方法
    }
};

class Derived : public Base {
public:
    void derivedMethod() {
        // 派生类方法
    }
};
```
