# OS 分段

OS看到的是虚拟地址空间

```shell
process virtual address space
+----------------------+
| stack                |
+----------------------+
| mmap area            |
+----------------------+
| heap                 |
+----------------------+
| data / bss           |
+----------------------+
| text / code          |
+----------------------+
```

## 2. 段页式OS管两层结构

段页式可以抽象成：

```shell
进程
  -> 段表
      -> 每个段对应一个页表
          -> 页表项映射到物理页框
```
