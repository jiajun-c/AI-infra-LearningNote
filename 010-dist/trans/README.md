# 分布式转置

场景是这样，我们有一个权重矩阵其大小为 `[M, K]`，卡的数量为N，每个卡上保存有 `[M/N, K]`这样大小的权重，将其本地进行一个转置，变为 `[K, M/N]`，然后再对其进行切分 `[K/N, M/N]`，然后通过all2all将其他的卡上。本身a2a我们就是可以把他看作是一次转置的操作

```shell
A2A (All-to-All) 示意图 — 4卡场景

转置前 (每卡持有行分块):
┌──────────────────────────────────────┐
│  GPU0: [chunk_00 | chunk_01 | ...]   │
│  GPU1: [chunk_10 | chunk_11 | ...]   │
│  GPU2: [chunk_20 | chunk_21 | ...]   │
│  GPU3: [chunk_30 | chunk_31 | ...]   │
└──────────────────────────────────────┘

All-to-All 通信 (每卡发送第j块给GPUj):
         GPU0   GPU1   GPU2   GPU3
GPU0 --> [00]   [01]   [02]   [03]
GPU1 --> [10]   [11]   [12]   [13]
GPU2 --> [20]   [21]   [22]   [23]
GPU3 --> [30]   [31]   [32]   [33]
          ↓      ↓      ↓      ↓
        col0   col1   col2   col3

转置后 (每卡持有列分块):
┌──────────────────────────────────────┐
│  GPU0: [chunk_00 | chunk_10 | chunk_20 | chunk_30]  │
│  GPU1: [chunk_01 | chunk_11 | chunk_21 | chunk_31]  │
│  GPU2: [chunk_02 | chunk_12 | chunk_22 | chunk_32]  │
│  GPU3: [chunk_03 | chunk_13 | chunk_23 | chunk_33]  │
└──────────────────────────────────────┘

原矩阵 [M, K] 按行分块 → A2A → 按列分块 [K, M/N]
本质：矩阵转置 = All-to-All 通信
```
